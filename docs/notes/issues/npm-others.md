## 概述

1. http-server 只可以下载文件，不可以下载文件夹，看看可不可以加一下这个功能。
2. [dufs](https://github.com/sigoden/dufs) 也是只可以下载文件，不可以下载文件夹。
3. Live Server vscode插件也是只可以下载文件，不可以下载文件夹。点击下载时，可以在服务器生成压缩zip文件，然后下载完成后再删除。或是保留，等下一次下载时，将过期zip删除。
